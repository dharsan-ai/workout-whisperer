# Workout Whisperer

Workout Whisperer is a simple fitness web app that helps users get workout suggestions, track exercises, and follow a beginner-friendly workout plan.

## What It Does

- Shows daily workout suggestions
- Lists exercises with reps, sets, and category
- Helps users follow a simple workout plan
- Shows beginner-friendly fitness guidance
- Tracks sample workout progress

## Core Screens

- Dashboard screen: Shows today's workout and quick stats
- Workout Plan screen: Displays suggested exercises
- Exercise List screen: Shows exercise name, reps, and category
- Progress screen: Shows completed workout progress

## Tech Stack

- React
- Vite
- JavaScript
- CSS
