# Workout Whisperer

Workout Whisperer is a simple fitness web app that helps users get workout suggestions, manage workouts, and follow a beginner-friendly fitness plan.

## What It Does

- Provides simple login/logout authentication
- Suggests workouts based on user goal and fitness level
- Lets users add new workouts
- Lets users edit existing workouts
- Lets users delete workouts
- Shows a list of saved workouts

## Core Screens

- Login screen: Allows a user to enter their name and access the app
- Dashboard screen: Shows the AI workout suggestion controls
- Workout Manager screen: Allows users to create, read, update, and delete workouts
- Recommended Plan screen: Shows suggested exercises based on goal and level

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Authentication

The app includes simple client-side authentication. A user enters their name to log in and can log out from the dashboard.

## CRUD Entity

The app includes a Workout CRUD entity. Users can create, read, update, and delete workout items.

## Deployment

The repo includes `vercel.json` for deployment configuration.
